# Cubos Backend

## API REST criada para responder o desafio backend da Cubos

##### Instruções (yarn)

1. Instalar dependências

```bash
yarn
```

2. Iniciar servidor

```bash
yarn start
```

3. Executar testes

```bash
yarn test
```

###### OBS: As requisições devem ser feitas na porta 3333

Exemplo: GET http://localhost:3333/regras
